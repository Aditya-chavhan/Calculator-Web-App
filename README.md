# 🧮 Calculator Web App

A simple and responsive **calculator web application** built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations with a clean and colorful UI.

---

## 🚀 Features

* ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division
* 🧠 Supports parentheses `( )` and percentage `%`
* 🧹 Clear button to reset calculations
* ⚡ Instant result using `=`
* 🎨 Attractive UI with different button colors
* 📱 Responsive design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Logic and functionality

---

## 📂 Project Structure

```
📁 Calculator
│── index.html      # Main structure
│── style.css       # Styling
│── script.js       # Functionality
```

---

## ⚙️ How It Works

* Buttons are captured using JavaScript event listeners
* User input is displayed on the screen
* Expression is evaluated using `eval()` function
* Clear (`C`) resets the screen
